# Design System — Legend Booking

Это единая система дизайна для продукта Legend Booking.

Она описывает:
- визуальные принципы
- компоненты
- цвета
- типографику
- поведение UI

---

## Принципы

- B2B-first (упор на читаемость)
- минимализм
- плотные интерфейсы (data-heavy UI)
- приоритет информации над декором
- быстрый сканинг данных

---

## Структура интерфейса

- Sidebar navigation (левое меню)
- Top bar (контекст страницы)
- Main content (основная зона)
- Right panel (опционально: детали)

---

## UI уровни

- L1: Dashboard (агрегированные данные)
- L2: Tables (основная работа)
- L3: Forms (создание/редактирование)
- L4: Modals (подтверждения/быстрые действия)
