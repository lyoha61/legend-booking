# Legend Booking
## Описание
Legend Booking — закрытая система краткосрочного бронирования апартаментов внутри одного жилого комплекса

Система управляет доступностью квартир, бронированиями и взаимодействием между:

управляющей компанией
собственниками
гостями

Цель системы — централизовать бронирование внутри ЖК и убрать зависимость от внешних маркетплейсов

---
## Основные функции
- поиск доступных апартаментов по датам
- управление доступностью квартир
- создание и управление бронированиями
- контроль правил проживания и ограничений
- аналитика для собственников

---
## Роли

### Гость
- поиск доступных апартаментов
- просмотр деталей квартиры
- бронирование

### Собственник 
- добавление и управление квартирами
- настройка правил (цены, ограничения, теги)
- просмотр доходов
