# Legend Booking
## Описание
Legend Booking — это веб-платформа для бронирования жилья, где пользователи могут искать и бронировать, размещенные объекты, а владельцы — создавать и управлять своими объявлениями.

Для владельцев предусмотрена панель управления (dashboard), которая позволяет контролировать объекты, отслеживать бронирования и анализировать финансовые показатели с помощью графиков доходов и расходов.

## Реализованный функционал (бэк и фронт)

- Показ всех объявлений
- Показ деталей объявления

## Стек
### Backend
- Java + Spring
- PostgreSQL
- JWT Auth

### Frontend
- React + Vite + TypeScript
- Feature-based архитектура
- Tailwind CSS
- React Router
- Zustand
